# 挖经历问题集:gap 期重启

**适用对象**:有 gap(职业空档期)需要重启求职的人。gap 原因不限——
- 考研 / 考公 / 考博失败后求职
- 备孕 / 育儿 / 照顾家人
- 长期病休 / 治疗
- 自我探索 / gap year
- 创业失败
- 长期失业 / 频繁跳槽中间空档
- 海外移民 / 签证空档

**核心原则**:**只问事实,不问动机 / 规划 / 感受**。"为什么 gap"、"心态怎样"、"未来怎么走"——这些不进简历。简历只需要展示:**gap 时间段 + gap 期实际产出**(自学 / 证书 / 副业 / 项目 / 公益)。

**使用方式**:按轮次问,每轮答完才进下一轮。

---

## 第 1 轮:Gap 时间段 + 类别

1. 你 gap 多久?(具体起止时间——这是必须写在简历时间线上的事实)
2. Gap 原因属于哪类?(只选大类,不需要解释)
   - 备考 / 升学
   - 健康 / 家庭照护
   - 育儿
   - 创业 / 副业
   - 旅行 / 自我探索
   - 待业
   - 海外签证 / 移民空档
   - 复合原因
3. 你**之前的职业方向**?(eg 互联网产品 / 银行柜员 / 教师)
4. **重启后想回原方向还是新方向**?(如新方向,叠加用 转行 问题集补)

---

## 第 2 轮:Gap 前职业经历

1. Gap 前最后一份工作?(公司 / 岗位 / 任期)
2. 在最后那份工作里**最自豪的 3 个成就**?(具体业务背景 + 量化结果)
3. 之前的总工作年限?换过几家公司?(简历时间线需要)
4. 你之前的**学历背景**?(学校 / 专业 / 学位)

---

## 第 3 轮:Gap 期实际产出(关键!)

> 这一节决定 gap 简历能不能成立——必须挖出**事实性的具体产出**

### 如果是育儿 / 父母 / 病亲照护
1. 期间你**协调过的资源**?(医院 / 学校 / 政府 / 保险——具体动作)
2. 学到的**硬技能**?(医疗知识 / 教育心理学 / 法律常识 / 财务规划)
3. **管理过的预算**?(具体金额)

### 如果是学习 / 备考
1. 备考具体内容?(全日制 / 在职?多久?)
2. 通过了什么科目 / 拿了什么证?(具体名称 + 时间)
3. 自学了哪些"备考之外"的技能?(语言 / 工具 / 编程)

### 如果是创业 / 副业
1. 做过什么项目?方向?
2. 用户数 / 收入 / 团队规模?(具体数字)
3. 即使失败,有什么可量化的成果?(eg "运营 3 万粉公众号"、"半年完成 80 万营收")

### 如果是自我探索 / 旅行
1. 去过哪些地方?多长时间?
2. 期间认识了什么人 / 加入什么社群 / 做过什么 volunteer?
3. 学了什么语言 / 文化技能?
4. 写过 / 输出过什么?(博客 / 视频 / 公众号 / 书籍——具体数据)

### 如果是健康
1. 治疗 / 康复期间**保持 / 培养的能力**?
2. 学了什么新东西?(阅读 / 在线课程 / 兴趣——具体)

---

## 第 4 轮:Gap 期主动建设(直接写进简历)

1. **在线课程**:Coursera / Udemy / 国内得到 / 极客时间 / 慕课等?(具体)
2. **专业证书**:gap 期拿到的(CFA / CPA / 教师证 / 法考 / NLP 等)
3. **副业 / 自由职业**:接过私单 / 写作 / 翻译 / 设计 / 咨询?
4. **个人项目**:GitHub / 公众号 / 小红书 / 抖音?(具体数据)
5. **志愿者 / 公益**:长期参与的志愿组织?
6. **行业关注**:gap 期间持续关注新方向 / 行业?(具体公众号 / 群 / podcast)

---

## 输出阶段

整理成 **`outputs/<用户>-素材库.md`**(纯事实):

```markdown
# <用户> Gap 重启素材库

## 时间线(简历必备)
- Gap 前工作:[公司 + 岗位 + 起止时间]
- Gap 期:[起止时间 + 大类]
- 现在:[重启求职中,具体何时开始]

## Gap 前职业经历
- [公司 + 岗位 + 任期 + 量化成就]

## Gap 期产出(显式写到简历,不留白)
- ⭐ 至少 2-3 条事实:
  - [证书 X]:gap 期 N 月获得
  - [自学 Y]:完成 Z 个课程 / 拿到 W 证书
  - [副业 / 项目]:用户数 / 收入 / 影响
  - [公益 / 长期志愿]:组织 + 时长 + 角色

## 技能 / 工具(包括 gap 期新增)
- 原岗位技能矩阵
- gap 期新增

## 自我评价拟句(纯事实)
- "[X 年] [原行业] 经验,gap N 月期间完成 [证书 1 + 自学 2 + 副业 3],目标重启 [新方向 / 原方向]"

## 缺失项 + 短期建议
- ⚠️ 如果 gap 期产出确实少 → 建议接下来 [具体动作] 补 [具体经历]
```

> **注意**:简历不写"我 gap 是因为...","我现在心态已经..."。这些是面试问题。简历只展示:gap 时间段 + gap 期具体做了什么(证书 / 自学 / 副业 / 项目)。

把素材库写好后,**自动进入下一步:重组**。
