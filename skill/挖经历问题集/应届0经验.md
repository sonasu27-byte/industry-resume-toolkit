# 挖经历问题集:应届 0 经验

**适用对象**:在校大学生 / 应届毕业生,无正式实习经历,或只有 ≤1 段不算正式的实习 

**核心原则**:**只问事实,不问动机 / 规划 / 感受**。你不是真的"没经历",是不知道自己做过什么算得上。这个清单帮你**系统性挖事实**——挖出来的内容直接进简历。

**使用方式**:按轮次问,每轮答完才进下一轮。每轮 4-6 个核心问题。

---

## 第 1 轮:教育背景

1. 你的学校 / 专业 / 学位是?在读还是已毕业?
2. GPA / 加权平均分 / 专业排名 / 年级排名(如知道)?
3. **你最自豪 / 最有收获的 3 门核心课**?分别学了什么?成绩?
4. 毕业设计 / 论文做什么?指导老师?核心方法?
5. 拿过哪些奖学金 / 荣誉?(国奖 / 励志奖 / 校级 / 院级)
6. 有没有交换 / 暑校 / 海外课程?

---

## 第 2 轮:课程项目

> 0 经验用户**最被低估的就是课程项目**——招聘官眼里好的课程项目 ≈ 实习

1. 你做过的**最复杂 / 时间最长**的课程项目是什么?
2. 这个项目是几人组?**你具体负责什么模块**?
3. 用了什么工具 / 方法论 / 模型?
4. **项目处理了多少数据 / 服务多少用户 / 覆盖多大范围**?
5. 最后的交付物是什么?报告 / 代码 / 设计稿 / 模型?有没有被采纳?
6. 这个项目有没有获奖 / 被推优 / 被引用?

---

## 第 3 轮:实习经历(哪怕只有几天 / 1 段)

1. 你做过任何实习吗?哪怕只有几天 / 一周?
2. 实习单位是什么类型(公司规模 / 行业)?
3. 你的具体岗位 / Title?
4. 实际做了什么动作?(写邮件 / 整理表格 / 跑数据 / 接电话 / 跟会议 / 做 PPT...)
5. 有没有独立完成的事 / 独立交付的物?

---

## 第 4 轮:校园经历(社团 / 学生组织 / 班委 / 志愿者)

1. 你在校期间担任过什么职务?(学生会 / 团委 / 班委 / 社团骨干 / 班长 / 寝室长)
2. 任期多长?具体管理 / 负责什么?
3. 任期内**主导或深度参与**过哪些活动?(迎新 / 晚会 / 比赛 / 公益 / 沙龙 / 讲座)
4. 这些活动你具体的角色?协调多少人 / 多大预算 / 多大规模?
5. 有没有做过志愿者?长期还是一次性?服务对象?
6. 有没有参与过校园媒体 / 自媒体 / 公众号 / 校园记者?
7. 三下乡 / 社会实践 / 暑期公益项目?

---

## 第 5 轮:比赛 / 个人项目 / 自媒体

1. **学科竞赛**:数学 / 物理 / 英语 / 数模 / 编程 / ACM?
2. **商业 / 创业类比赛**:挑战杯 / 互联网+ / 创青春 / 大创 / 各品牌商赛?
3. **自己运营的项目**:公众号 / 小红书 / 抖音 / B 站 / GitHub / 博客?有多少粉 / 互动数据?
4. **副业 / 兼职**:家教 / 翻译 / 设计 / 写作 / 代购 / 校园代理?
5. **自学过什么**:在线课程 / 证书 / 编程 / 设计 / 语言?
6. **家庭 / 个人项目**:帮家里做什么生意 / 搞研究 / 做创作?

---

## 第 6 轮:技能 / 工具 / 语言 / 证书

1. 英语水平?CET-4 / CET-6 分数?雅思 / TOEFL?其他外语?
2. **办公软件**熟练度?(Excel 函数 / 透视表 / VBA?PPT 设计?)
3. 专业工具?(数据分析:Python/SQL/SPSS;设计:Figma/PS;视频:剪映/PR)
4. 拿过什么证书?(CPA / CFA 备考 / 教师资格 / 普通话 / 急救...)

---

## 输出阶段

问完所有轮次,整理成 **`outputs/<用户>-素材库.md`**(纯事实,直接可写进简历):

```markdown
# <用户> 素材库

## 教育背景
- [学校 / 专业 / GPA / 排名 / 核心课程 / 论文]

## 课程项目
- 项目 1:[背景 + 你的角色 + 工具 + 量化 + 交付物]
- 项目 2:...

## 实习经历(如有)
- [按时间倒序,公司 + Title + 动作 + 交付物]

## 校园经历
- 学生干部:[职务 + 任期 + 管理范围 + 主导活动 + 规模]
- 志愿者 / 公益:[组织 + 时长 + 角色]
- 校园媒体 / 自媒体:[平台 + 数据]

## 比赛 / 个人项目 / 自媒体
- [比赛名称 + 排名 + 角色]
- [个人项目数据]
- [副业:类型 + 时长 + 量化]

## 技能 / 工具 / 证书
- 语言:[英语 + 第二外语]
- 工具:[精通 / 熟练 / 掌握 三档分类]
- 证书:[已拿到的]

## 缺失项(需用户确认或后续补)
- ⚠️ [项目 X 的数据]:用户没记清 → 投简历前必须补
- ⚠️ [实习 Y 的时间]:不确定 → 需要补
```

把素材库写好后,**自动进入下一步:重组**。
