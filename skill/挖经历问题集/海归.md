# 挖经历问题集:海归 / 海外背景重写国内版

**适用对象**:
- 海外硕士 / 博士回国求职
- 海外本科回国求职
- 海外有工作经历回国
- 国内本科 + 海外硕士的"小海归"

**核心原则**:**只问事实,不问动机 / 规划 / 感受**。"为什么回国"、"未来在哪发展"——这些是面试问题。简历只需要展示:**你海外学了 / 做了什么 + 这些事实怎么在国内简历里翻译呈现**。

**附加价值**:海归是 v0.2 最适合**双语输出**的用户群。

**使用方式**:按轮次问,每轮答完才进下一轮。

> 注:用户的"目标岗位 + 行业 + 企业性质 + 简历语言"已在 `/诊断` 流程里捕获。

---

## 第 1 轮:教育背景(海外 + 国内)

1. 国内本科 / 海外硕士(博士)各是什么学校 / 专业?
2. 学校排名 / QS / THE 排名(如知道)?
3. 学位类型(MS / MA / MBA / PhD / MEng / LLM)?
4. 海外是几年制?是 taught 还是 research?
5. 英语水平?雅思 / TOEFL / GMAT / GRE 分数?
6. 第二外语(法 / 德 / 西 / 日 / 韩 / 阿)?

---

## 第 2 轮:海外学术深挖

1. **专业核心课程**:最有收获的 3-5 门?分别学了什么?
2. **课程项目(Coursework)**:有没有标志性的 group / individual / case study?
3. **毕业论文 / 设计 (Dissertation / Thesis)**:题目?导师?方法论?核心结论?
4. **学术发表 / 会议**:有没有 paper / poster / 会议 presentation?
5. **学院活动 / Society**:加入什么学生社团 / professional society?担任职务?
6. **奖学金 / 荣誉 / Honors**:Dean's List / Distinction / Merit?

---

## 第 3 轮:海外实习 / 工作经历

> **关键点**:海外实习要让**国内 HR 看懂含金量** —— 公司中英文都标 + 行业 + 项目类型

1. 海外实习 / 工作的公司是?(中英文名都告诉我)
2. **公司在国内的认知度**?(eg "Mars" 国内不熟,说"和德芙 / 士力架 / M&M's 的母公司,等同国内大型快消" HR 就懂了)
3. 你的 title(英文原文 + 中文意译)?
4. 主要项目 / 工作内容?客户?项目规模?
5. 量化产出?(用国际单位 + 国内可读单位 都给)

---

## 第 4 轮:Extracurriculars / Leadership

> 海外简历重视的项,国内简历对应是"校园经历 / 课外活动"

1. 担任过什么 student leadership 角色?(Student rep / society president / club leader)
2. 参与过什么 case competition / business plan competition / hackathon?
3. 做过什么 volunteer / community service / pro-bono?
4. 跨文化经历?(参与海外学生组织 / 多元文化项目)
5. 自己 launch 的项目?(创业 / 公众号 / 自媒体)

---

## 第 5 轮:技能 + 工具 + 国际化资产

1. 英语水平 self-assessment(口语 / 书面 / 商务能力分别什么级别?)
2. **工具栈**:海外学校 / 公司用的(R / Stata / Eviews / Bloomberg Terminal / Salesforce)?
3. **跨文化能力**实证:在海外跟多少 nationality 的人合作过?有没有用第二语言完成 deliverable?
4. 国际化资产:出过书 / 主持过国际活动 / 多语言访谈?

---

## 第 6 轮:中英双语简历策略(只问事实选择)

1. 你需要简历**几个版本**?
   - 仅中文(纯国内 industry 求职)
   - 仅英文(外企 / 海外岗位)
   - **中英双版本**(既投国内又投外企 / 海外)
2. 中文版主要投什么类型公司?(决定套哪份 patterns/企业性质)
3. 英文版主要投什么类型公司?(决定套外企 patterns 的哪个细分)
4. 简历目标长度?(中文 1-2 页;英文 1-2 页国际惯例)
5. 你**回国时间**多久了?(只是事实,不问"为什么回")

---

## 输出阶段

整理成 **`outputs/<用户>-素材库.md`**:

```markdown
# <用户> 海归素材库

## 教育背景(中英双语)
- 海外:[校名中英 + QS 排名 + 学位类型]
- 国内本科:[同理]

## 海外经历(国内可读版 + 原始英文版)
- 公司中英文名 + 国内对标
- title 中文意译
- 量化(双单位)

## Extracurriculars / Leadership
- (国内简历叫"校园经历")

## 国际化资产
- 语言 / 跨文化 / 第二专业

## 简历版本计划
- 中文版:投 [公司类型],用 [patterns]
- 英文版:投 [公司类型],用 patterns/企业性质/外企.md
- 回国 [N 月]

## 自我评价拟句(纯事实)
- "海外硕士(XX 大学,YY 方向)+ 国内本科,X 段海外 / 国内实习,熟悉中英双语环境,目标 [岗位] @ [企业性质]"

## 国内适配缺口(事实判断,不问感受)
- ⚠️ 海外术语 X 国内不通用 → 简历里换成 Y
- ⚠️ 学校认知度 → 加排名说明
- ⚠️ 公司认知度 → 加国内对标括号说明
```

> **注意**:简历不写"我回国是因为..."、"我未来想..."。这些是面试问题。简历只展示事实:海外学了什么 + 做过什么 + 拿过什么证 + 会什么语言。

把素材库写好后,**自动进入下一步:重组**。
